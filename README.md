# Portfolio

`sl820` 的个人作品集 — 双轨：设计 + 学术研究。

## 技术栈

- [Astro 5](https://astro.build) · 静态优先，零 JS 默认
- TypeScript strict
- Tailwind CSS v4（CSS-first config）
- MDX + Content Collections（zod schema）
- GitHub Pages 部署（Actions + pnpm）

## 开发

```bash
pnpm install
pnpm dev            # localhost:4321
pnpm build
pnpm preview
```
