公共图书馆研学服务的理论起源与发展脉络。中英双轨综述，最终纳入 50 + 40 = 90 篇。

研究问题

公共图书馆的「研学服务」在中国 2017 年后兴起，其理论基础是什么？ 与英美 Reading Promotion / Library Instruction 之间的关系如何？

方法

PRISMA 2020 完整流程。中英双轨、字段代码适配、阶段 CSV 化追溯。

时期划分

5 时期，50 篇最终清单分布：

  • P1 萌芽期（<2000）：5 篇
  • P2 奠基期（2000–2010）：10 篇
  • P3 发展期（2010–2015）：13 篇
  • P4 扩展期（2015–2020）：12 篇
  • P5 成熟期（2020–2026）：10 篇

核心文献

  • 沈彤 2004 / 白帅敏 2015 / 颜虹 2003 / 朱小玲 2009（中文 P1 补充）
  • 国际图联 IFLA 报告 / ALA 公共图书馆手册

关键发现

「研学」一词的本土化，与西方 Information Literacy / Learning Experience 概念既重叠又分歧。 中国语境的「研学」更强调 校外教育补充，西方更强调 终身学习能力。
